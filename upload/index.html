# This is a simple Bash script that performs the following actions:

Greets the current user by printing "Hello" followed by the username.
Prints the current date and week number.
Prints a statement: "This here is a Script".
Prints a message: "these users are Currently using system".
Lists the unique usernames of users currently logged into the system. 
This is achieved by using the w command to get a snapshot of the current system status, 
cut to extract the usernames, grep -v to exclude the header line, and sort -u to sort and remove duplicates.
