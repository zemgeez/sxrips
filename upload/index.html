# This is a simple Bash script that performs the following actions:

Greets the current user by printing "Hello" followed by the username.
Prints the current date and week number.
Prints a statement: "This here is a Script".
Prints a message: "these users are Currently using system".
Lists the unique usernames of users currently logged into the system. 
This is achieved by using the w command to get a snapshot of the current system status, 
cut to extract the usernames, grep -v to exclude the header line, and sort -u to sort and remove duplicates.



Prints the username of the current user.
Prints a message indicating that a new file is being created along with its backup.
Creates a new file named newfile.txt using the touch command.
Creates a backup of newfile.txt named newfile_backup.txt using the cp command.
Prints a message indicating that the operation is done and the new file contains system information.
Writes the operating system information to newfile.txt using the uname -a command.
Appends the CPU information to newfile.txt using the lscpu command.
Appends the memory information to newfile.txt using the free -h command.
